# R 語言統計工具入門：用 AI 學會做臨床研究統計

這是一個 Quarto 書籍項目，教授如何使用 AI 輔助學習 R 語言進行臨床研究統計分析。

## 在線閱讀

📖 **書籍網址**: https://htlin222.github.io/2025-12-05-R-Basics/

📁 **GitHub 倉庫**: https://github.com/htlin222/2025-12-05-R-Basics

## 項目結構

- `index.qmd` - 封面頁
- `part1.qmd` - 第一部分：你今天就會寫程式
- `part2.qmd` - 第二部分：讀取你的資料
- `part3.qmd` - 第三部分：產出你的 Table 1
- `part4.qmd` - 第四部分：畫出論文等級的圖
- `part5.qmd` - 第五部分：統計檢定
- `part6.qmd` - 第六部分：整合與收尾
- `appendix.qmd` - 附錄：給講師的備註
- `patient_data.csv` - 示例數據
- `_quarto.yml` - Quarto 配置文件

## 構建說明

### 安裝依賴

確保您已安裝：

- [Quarto](https://quarto.org/docs/get-started/)
- R 和 RStudio

### 構建書籍

```bash
quarto render
```

這將生成 HTML 和 PDF 版本的書籍在 `_book/` 目錄中。

### 預覽書籍

```bash
quarto preview
```

這將啟動一個本地服務器來預覽書籍。

## 課程目標

本課程的目標是教授醫護人員如何：

- 使用 AI（ChatGPT/Claude）輔助寫 R 程式
- 產出論文級別的統計表格（Table 1）
- 創建專業的統計圖表
- 執行基本的統計檢定
- 建立可重複使用的分析範本
